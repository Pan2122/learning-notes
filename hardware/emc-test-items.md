# EMC 测试项目速查：RE、CE、RI 与常见抗扰度

> 简介：梳理 EMC 中发射类与抗扰度类测试的方向、缩写、典型现象和常见应用场景，方便快速判断“设备往外干扰别人”还是“外界干扰打进设备”。

## 1. 先记住大框架

EMC 可以先分成两大类：

```text
EMC = EMI + EMS
```

| 类别 | 英文 | 中文 | 关注点 |
| --- | --- | --- | --- |
| EMI | Electromagnetic Interference | 电磁干扰 / 电磁发射 | 设备会不会干扰别人 |
| EMS | Electromagnetic Susceptibility | 电磁抗扰度 | 外界干扰来了，设备能不能正常工作 |

如果是在问“测设备往外辐射值”，一般指：

```text
RE：Radiated Emission
中文：辐射发射 / 辐射骚扰
```

也有人口头叫：

```text
辐射发射测试
辐射骚扰测试
空间辐射测试
EMI 辐射测试
```

它测的是：设备工作时，通过空间向外辐射出去的电磁噪声有没有超标。

## 2. EMI：发射类测试

发射类测试关注的是：

```text
我的设备会不会把噪声发出去，影响别人？
```

### 2.1 RE：Radiated Emission，辐射发射

RE 就是“设备往外辐射值”对应的项目。

它测设备通过空气向外辐射的电磁噪声，典型单位是：

```text
dBμV/m
```

常见测试距离：

```text
3 m / 10 m
```

常见频段大致为：

```text
30 MHz ~ 1 GHz
1 GHz ~ 6 GHz / 18 GHz
```

具体频段要看产品适用标准。

| 频段 | 常见来源 |
| --- | --- |
| 几十 MHz | MCU 时钟、晶振、DC/DC 开关频率谐波、线缆共模电流 |
| 100~300 MHz | 排线、IO 线、外接线缆变成天线 |
| 300 MHz 以上 | 高速边沿、USB、SDIO、SPI、MIPI、LVDS、时钟谐波 |
| 1 GHz 以上 | 高速数字接口、无线模块、强边沿信号 |

对 STM32、TDM、A2B、TF 卡这类系统，如果 RE 超标，优先怀疑：

```text
1. 外接线缆变成天线
2. 时钟线或高速边沿信号回流路径差
3. 板上 GND 不连续
4. 共模电流从接口线缆跑出去
5. DC/DC 或晶振谐波
```

### 2.2 CE：Conducted Emission，传导发射

CE 测的是设备通过电源线、信号线传出去的干扰，常见单位是：

```text
dBμV
```

通常通过 LISN 或耦合网络测量。典型噪声来源包括：

```text
DC/DC 开关噪声
电机驱动噪声
MCU / 数字电路回流噪声
隔离电源噪声
```

CE 和 RE 的区别：

| 项目 | 中文 | 干扰路径 |
| --- | --- | --- |
| CE | 传导发射 | 沿电源线 / 线缆传出去 |
| RE | 辐射发射 | 通过空间辐射出去 |

一句话：

```text
CE 看线上的噪声；
RE 看空气中的噪声。
```

### 2.3 Harmonic Current，谐波电流

谐波电流主要针对接交流市电的设备。

它测设备从 AC 电源吸电流时，会不会产生过多低频谐波，例如：

```text
50 Hz 基波
100 Hz / 150 Hz / 250 Hz / 350 Hz 等谐波
```

常见标准是 IEC 61000-3-2。如果设备只是低压 DC 供电，很多时候不涉及这个项目。

### 2.4 Voltage Flicker，电压波动和闪烁

电压波动和闪烁同样主要针对接交流市电的设备。

它测设备工作时电流变化是否过大，以至于导致电网电压波动，进而让灯具等负载出现可感知的闪烁。常见标准是 IEC 61000-3-3。

## 3. EMS：抗扰度类测试

EMS 是反过来测：

```text
外界干扰打到设备上，设备还能不能正常工作？
```

### 3.1 RS / RI：辐射抗扰度

不同公司叫法可能不完全一样：

```text
RS：Radiated Susceptibility
RI：Radiated Immunity
中文：辐射抗扰度
```

它用天线对设备扫频辐射，让设备处在强电磁场里，看功能是否异常。

常见单位：

```text
V/m
```

常见等级示例：

```text
3 V/m
10 V/m
30 V/m
```

对传感器采集板来说，RI 测试时可能看到：

```text
1. 传感器数据跳变
2. I2C / SPI / TDM 通讯异常
3. MCU 死机或复位
4. TF 卡写入失败
5. 串口打印乱码
6. 某个频点数据明显偏移
```

RE 和 RI/RS 的区别：

| 项目 | 中文 | 方向 |
| --- | --- | --- |
| RE | 辐射发射 | 设备向外辐射，干扰别人 |
| RI / RS | 辐射抗扰度 | 外界辐射打进来，考验设备自己 |

一句话：

```text
RE：我吵不吵别人？
RI：别人吵我，我扛不扛得住？
```

### 3.2 CS / CI：传导抗扰度

```text
CS：Conducted Susceptibility
CI：Conducted Immunity
中文：传导抗扰度
```

它把射频干扰通过线缆注入到设备里，看设备是否异常。

常见方式：

```text
CDN 注入
BCI 注入
电流钳注入
```

RI/RS 和 CS/CI 的区别：

| 项目 | 干扰怎么进来 |
| --- | --- |
| RI / RS | 通过空气辐射进来 |
| CS / CI | 通过电源线 / 信号线传导进来 |

如果传感器通过长线连接到 MCU，CS/BCI 这类测试也很关键。

### 3.3 ESD：静电放电抗扰度

ESD 模拟人手或金属物体接触设备时的静电放电。

常见形式：

```text
接触放电
空气放电
```

常见等级：

```text
±4 kV
±8 kV
±15 kV
```

常见打点：

```text
外壳
按键
连接器金属壳
USB 口
电源口
裸露测试点
```

常见问题：

```text
MCU 复位
USB 掉线
数据异常
触摸失灵
芯片损坏
```

### 3.4 EFT / Burst：电快速瞬变脉冲群

```text
EFT：Electrical Fast Transient
Burst：脉冲群
中文：电快速瞬变脉冲群抗扰度
```

它模拟继电器、电机、电感负载开关瞬间产生的一串高频尖峰干扰。

主要耦合到：

```text
电源线
信号线
控制线
```

常见问题：

```text
串口丢包
I2C 错误
MCU 误触发
复位
存储异常
```

对 MCU 板、传感器线和通信线来说，这个测试很有杀伤力。

### 3.5 Surge：浪涌抗扰度

```text
Surge Immunity
中文：浪涌抗扰度 / 雷击浪涌
```

它模拟雷击、电网切换、大功率设备启停导致的大能量冲击。

特点是：

```text
能量比 EFT 大很多
速度比 ESD 慢
破坏性更强
```

常见端口：

```text
AC 电源口
DC 电源口
长距离信号线
户外线缆
```

常见防护器件：

```text
TVS
压敏电阻 MOV
气体放电管 GDT
保险丝 / PTC
共模电感
滤波电容
```

### 3.6 Voltage Dips / Interruptions，电压暂降和短时中断

```text
电压暂降
短时中断
电压变化抗扰度
```

它模拟电源突然掉一下、低一下、断一下，看设备是否能保持或恢复正常。

示例：

```text
电压掉到 70%
电压掉到 40%
电压掉到 0%
持续 10 ms / 100 ms / 500 ms
```

重点观察：

```text
设备是否死机
数据是否丢失
是否能自动恢复
是否误动作
```

### 3.7 Power Frequency Magnetic Field，工频磁场抗扰度

```text
中文：工频磁场抗扰度
```

它模拟 50/60 Hz 大电流设备产生的低频磁场干扰。

对以下系统更敏感：

```text
磁传感器
霍尔传感器
电流采样
低频模拟前端
长环路 PCB
```

如果测的是加速度计，一般不是最核心项目；但如果系统里有磁传感器、霍尔或电流检测，就要关注。

## 4. 汽车电子常见 EMC 项目

如果产品偏汽车、车载传感器或车载 ECU，经常会看到这些名称：

| 项目 | 含义 |
| --- | --- |
| CISPR 25 CE / RE | 车载零部件传导 / 辐射发射 |
| ISO 11452-2 | 车载辐射抗扰度 |
| ISO 11452-4 BCI | 大电流注入抗扰度 |
| ISO 7637-2 | 电源线瞬态传导抗扰度 |
| Load Dump | 抛负载浪涌 |
| ESD ISO 10605 | 车载静电测试 |

汽车电子里还常见：

```text
ALSE 法：暗室天线照射
BCI：大电流注入
DPI：直接功率注入
TEM Cell：小型辐射抗扰度测试
```

## 5. 快速记忆表

### 发射类：设备干扰别人

```text
CE：Conducted Emission，传导发射
RE：Radiated Emission，辐射发射
Harmonic：谐波电流
Flicker：电压波动 / 闪烁
```

### 抗扰类：别人干扰设备

```text
ESD：静电
EFT / Burst：快速脉冲群
Surge：浪涌
RI / RS：辐射抗扰度
CI / CS：传导抗扰度
Dips：电压暂降 / 中断
PFMF：工频磁场
```

## 6. 容易混淆的缩写

| 缩写 | 全称 | 中文 | 方向 |
| --- | --- | --- | --- |
| CE | Conducted Emission | 传导发射 | 往外 |
| RE | Radiated Emission | 辐射发射 | 往外 |
| CS / CI | Conducted Susceptibility / Immunity | 传导抗扰度 | 往里 |
| RS / RI | Radiated Susceptibility / Immunity | 辐射抗扰度 | 往里 |
| ESD | Electrostatic Discharge | 静电放电 | 往里 |
| EFT | Electrical Fast Transient | 电快速瞬变脉冲群 | 往里 |
| Surge | Surge Immunity | 浪涌抗扰度 | 往里 |

最核心的一句话：

```text
测设备往外辐射：RE，辐射发射。
测设备抗外界辐射：RI / RS，辐射抗扰度。
测设备往电源线传噪声：CE，传导发射。
测设备抗线缆注入干扰：CS / CI，传导抗扰度。
```
